# Is Touching

## Description

Returns a boolean on whether or not two provided objects are touching.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Part1 (Object)

You give the path to an object.

* [x] Can use Variable

```
workspace.Baseplate
```

#### Part2 (Object)

You give the path to an object.

* [x] Can use Variable

```
workspace.Part
```

## Outputs

#### Result

You give the block the variable you want the result to be saved to.

```
AreTheyTouching
```
